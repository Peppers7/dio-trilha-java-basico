# ContaBanco

Projeto simples em Java que simula a criação de uma conta bancária via terminal.

## Funcionalidades

- Recebe dados do usuário pelo terminal:
  - Nome do cliente
  - Número da agência
  - Número da conta
  - Saldo da conta
- Exibe uma mensagem de confirmação com as informações cadastradas.
